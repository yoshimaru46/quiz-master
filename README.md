# README
This is Simple Quiz App with Rails & React

## DEMO

[DEMO](/DEMO.md)

## Versions

- ruby: 2.5.0-rc1
- rails: 5.1.6 

## Setup
1. Get the code.

    `% git clone git@github.com:yshr446/quiz-master.git`

2. Setup your environment.

    `% bin/setup`

2. Install modules

    `% npm install`

3. Start Foreman.

    `% foreman start`

4. Verify that the app is up and running.

     `% open http://localhost:3030`
